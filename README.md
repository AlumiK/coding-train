# 三阶魔方

项目基于 [Processing](https://processing.org/) 语言制作。

## 操作说明

使用空格键打乱魔方。

使用以下按键旋转魔方。

| 面颜色 | 方向 | 按键 |
| ----- | ----- | ----- |
| 白色 | 顺时针 | `U` |
| 白色 | 逆时针 | `u` |
| 黄色 | 顺时针 | `D` |
| 黄色 | 逆时针 | `d` |
| 橙色 | 顺时针 | `L` |
| 橙色 | 逆时针 | `l` |
| 红色 | 顺时针 | `R` |
| 红色 | 逆时针 | `r` |
| 蓝色 | 顺时针 | `B` |
| 蓝色 | 逆时针 | `b` |
| 绿色 | 顺时针 | `F` |
| 绿色 | 逆时针 | `f` |
